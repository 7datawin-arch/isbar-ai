## ✨ Features

### 🤖 AI Chatbot + Knowledge Base
- 25+ topics about AI, courses, services
- Typewriter effect responses
- Quick reply buttons
- Chat history saved locally
- Somali language support

### 📝 Quiz + Gamification
- **28 questions** across 4 categories (AI, Technology, Somali, General)
- **Timer** — 30 seconds per question
- **Points** — +10 correct, -5 wrong, speed bonus
- **10 Levels** — from Cusub to Master AI
- **4 Badges** — 🌟 Bilow, ⚡ Dhexe, 🔥 Khabiir, 👑 Master AI
- **Leaderboard** — top 20 players saved locally

### 📚 Free PDF Books
- AI Fundamentals (42 pages)
- ChatGPT-ga oo Fudud (38 pages)
- More coming soon
- Free for course students
